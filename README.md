## Project Overview
This project predicts whether a passenger survived the Titanic disaster using machine learning.

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Flask
- HTML/CSS

## Model Details
- Algorithm: Decision Tree / Logistic Regression
- Feature Scaling: StandardScaler
- Encoding: LabelEncoder / OneHotEncoder

## Workflow
1. Model trained in Jupyter Notebook
2. Model saved using Joblib
3. Flask app built in VS Code
4. Web UI created using HTML
